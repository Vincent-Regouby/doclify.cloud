# Plan Marketing Doclify.cloud

> Stratégie marketing complète pour le lancement et la croissance de Doclify.cloud

## Vue d'ensemble

**Doclify** est une plateforme SaaS de transcription médicale par IA, conçue spécifiquement pour les professionnels de santé français.

### Proposition de valeur

- **Gain de temps** : 2h/semaine économisées sur les tâches administratives
- **Souveraineté** : 100% des données hébergées en France
- **RGPD natif** : Conformité totale avec la réglementation européenne
- **Prix compétitif** : 75€/mois vs 150-200€ chez les concurrents

### Cible

| Segment        | Description                                              |
| -------------- | -------------------------------------------------------- |
| **Primaire**   | Médecins libéraux (généralistes, spécialistes, psy)      |
| **Secondaire** | Cabinets de groupe (3+ praticiens)                       |
| **Tertiaire**  | Cliniques et établissements de santé                     |
