## 4. SEO Technique

### Actions prioritaires

- [ ] Optimiser les balises title/meta description de chaque page
- [ ] Ajouter schema.org (SoftwareApplication, Product, FAQPage)
- [ ] Créer un sitemap.xml dynamique
- [ ] Optimiser Core Web Vitals (LCP, FID, CLS)
- [ ] Implémenter les balises hreflang (préparation internationalisation)

### Structure des URLs

```
/                           # Homepage
/fonctionnalites           # Features
/tarifs                    # Pricing
/a-propos                  # About
/contact                   # Contact
/blog/                     # Blog index
/blog/[slug]              # Articles
/alternative-dragon-medical # Landing page
/transcription-[specialite] # Pages par spécialité
```

### Balises meta par page

```html
<!-- Homepage -->
<title>Doclify - Transcription médicale IA pour médecins français</title>
<meta name="description" content="Gagnez 2h/semaine avec la transcription médicale automatique. 100% hébergé en France, conforme RGPD. Essai gratuit 14 jours.">

<!-- Features -->
<title>Fonctionnalités - Transcription médicale automatique | Doclify</title>
<meta name="description" content="Enregistrement mobile, transcription IA, résumés intelligents. Découvrez comment Doclify simplifie votre documentation médicale.">

<!-- Pricing -->
<title>Tarifs - Transcription médicale à partir de 65€/mois | Doclify</title>
<meta name="description" content="75€/mois pour médecins indépendants, 65€/mois pour cabinets. Sans engagement, essai gratuit 14 jours.">
```
