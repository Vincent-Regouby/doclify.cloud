## 3. Newsletter mensuelle

### Structure

1. **Édito** (3-4 lignes personnelles)
2. **Nouveautés produit** (1-2 features)
3. **Article du mois** (lien blog)
4. **Témoignage client**
5. **Astuce du mois**
6. **CTA** (essai ou upgrade)

### Calendrier

- **Envoi** : Premier mardi du mois, 8h30
- **Rédaction** : Semaine précédente
- **Validation** : Vendredi avant

### Template objet

```
[Prénom], les nouveautés Doclify de [Mois]
Ce que 200 médecins font différemment + 1 nouvelle fonctionnalité
[Mois] : transcription 30% plus rapide + votre astuce productivité
```

## 6. Bonnes pratiques

### Délivrabilité

- [ ] Configurer SPF, DKIM, DMARC
- [ ] Utiliser un domaine d'envoi dédié (ex: mail.doclify.cloud)
- [ ] Warmup progressif du domaine
- [ ] Nettoyer la base tous les 3 mois (bounces, inactifs)

### Conformité RGPD

- [ ] Double opt-in obligatoire
- [ ] Lien de désinscription visible
- [ ] Mention légale en footer
- [ ] Politique de confidentialité à jour

### A/B Testing

Tester systématiquement :
- Objets (2-3 variantes)
- Heures d'envoi
- Longueur des emails
- CTA (texte, couleur, position)
